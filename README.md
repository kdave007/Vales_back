# Vales Backend API

## Development Setup

### Prerequisites
- Node.js 18+
- Docker 20.10+

### Installation
```bash
npm install

GET /api/mock - Test mock endpoint



Build Image

    docker build -t vales-backend .

Run Container

    docker run -p 3000:3000 vales-backend

View Running Containers

    docker ps

Stop Container

    docker stop <container-id>
